Cross-Agent comparison

How the seven agents stack up. Same capability, different solutions. Use this page to pick the right one for your project — or steal the right idea for yours.

Note: Overmind is in a different category than the other six — it's an optimizer for agents, not a task-doing agent. We include it because its mechanisms (regression-aware acceptance, cross-run FailureRegistry, provenance scoring) are independently useful design patterns.

Design Philosophy — One Sentence Each

Pi — "If you want the agent to do something new, ask it to build it."
OpenClaw — "Pi, but on every messaging surface you live on."
Hermes — "Ship one Python class with every feature you can imagine, then teach it to improve itself."
OpenCode — "Treat your language server as the agent's compiler."
Deep Agents — "Compose your agent from middleware; let LangGraph hold the state."
Codex CLI — "Don't trust the model — let the OS enforce."
Overmind — "Every prompt change is a falsifiable claim. Make it pass the gates."
