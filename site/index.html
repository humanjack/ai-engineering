The Seven Agents

01

Pi

TypeScript · badlogic/pi-mono

Minimal self-extending coding agent. 4 built-in tools, tree-shaped sessions, hot-reloadable TypeScript extensions. Powers OpenClaw.

Read deep-dive →
02

OpenClaw

TypeScript · openclaw/openclaw

Multi-channel productization of Pi. Gateway daemon + 22+ messaging surfaces + 122 plugins + cron + Docker sandboxes.

Read deep-dive →
03

Hermes Agent

Python · NousResearch/hermes-agent

Maximalist Python harness with a self-improving skill curator. 16k LOC of agent engineering and a 7-day skill review cycle.

Read deep-dive →
04

OpenCode

TS + Go · sst/opencode

LSP-native coding agent. Edits wait for compiler diagnostics. Tree-sitter bash perms. Shadow-git snapshots. No vector DB.

Read deep-dive →
05

Deep Agents

Python · langchain-ai/deepagents

LangChain's opinionated harness. Middleware-composable. Sub-agents, summarization, large-result eviction, harbor evals.

Read deep-dive →
06

Codex CLI

Rust · openai/codex

One Op/EventMsg protocol, five frontends, three-layer OS sandboxes. Claude-Code-compatible hooks. First-class sub-agent mailboxes.

Read deep-dive →
07

Overmind

Python · overmind-core/overmind

The agent that optimizes other agents. Policy-driven best-of-N candidate generation in git worktrees. Regression-aware acceptance gates. Cross-run FailureRegistry.

⚡ Quick Pick — Which One Should I Use?

Need a minimal core?
Pi — 4 tools, build the rest yourself
Multi-channel personal assistant?
OpenClaw — 22+ messaging surfaces
Self-improving skills?
Hermes — `skill_manage` + curator
Code-editing agent with type-checking?
OpenCode — LSP feedback in the loop
Middleware-composable, with evals?
Deep Agents — LangGraph + `harbor`
Need real OS-enforced sandboxing?
Codex CLI — 3 layers per platform
Want to improve an existing agent?
Overmind — policy-driven optimizer with regression-aware gates
